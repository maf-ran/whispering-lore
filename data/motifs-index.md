# Motifs Index (ATU / ML Classification)

This file contains motif classifications used for folklore research.  
The agent must use this file to identify motif numbers, classify stories, and detect parallels.

---

## Purpose
- Provide a reference for ATU (Aarne–Thompson–Uther) tale types  
- Provide a reference for ML (Motif-Index of Folk Literature) motifs  
- Help classify creatures, stories, and legends  
- Improve cross-cultural comparison  
- Support motif-based searching
