# Aliases Dictionary (Name Variants & Synonyms)

This file contains alternative names, spellings, and regional variants for creatures, spirits, and folklore entities.  
The agent must use this file to detect duplicates, merge entries, and improve search accuracy.

---

## Purpose
- Normalize names across regions and languages  
- Detect duplicate entities  
- Improve search queries  
- Expand research coverage  
- Support historical and dialect spellings  

---

## Format
Each entry follows this structure:

### Entity Name
- Primary: <main name>
- Aliases:
  - <alias 1>
  - <alias 2>
  - <alias 3>
- Historical Spellings:
  - <old spelling 1>
  - <old spelling 2>
- Regional Variants:
  - <region>: <variant>
  - <region>: <variant>
- Notes:
  - <context or explanation>

## Rules for the Agent
- Always check this file before deciding if two names refer to the same entity.
- Use aliases to expand search queries.
- Use historical spellings for archival research.
- Add new aliases when discovered.
- Avoid creating duplicate datasets for entities with matching aliases.
